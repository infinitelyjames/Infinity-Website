Details

Revisioner is an app under development designed to aid revision by blocking potential distractions and distractive material. This includes but is not limited to website blocking, reminders (schedule website blocking to enable & disable), and Screen Savers.
Personalisation options include light/dark mode, background images & more...

FAQ

Why are some websites not blocked immediately?

Due to browser caching, the website may continue to load unblocked for up to ~10 minutes after enabling website blocking. Close all browser windows and re-open if the issue persists.

Why do I get an "Insecure Connection" error in Firefox and other web browsers?

We're effectively interrupting the connection and replacing the website with the blocked page. Your web browser is correct to warn you, since the resulting webpage is abnormal.

About

Revisioner is programmed in Python in conjunction with Qt (PyQt). Most windows utilise Qt features, although the loading splash screen is using tkinter. The app is exported using PyInstaller.
